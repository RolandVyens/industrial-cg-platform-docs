# Manuel de l'AIO-OCIO

Bienvenue dans le manuel d'utilisation officiel de l'**AIO-OCIO** (configuration All in 1 OpenColorIO).

## **Ordre de lecture recommandé**

1. [Configuration Générale des Logiciels](./general-softwares) — Intégration avec Maya, Nuke, Houdini, et configuration des variables d'environnement système.
2. [Configuration pour Blender](./blender-setup) — Surcharges personnalisées pour Blender et configuration du flux de travail Lin709.

---

> 💡 **Concept de conception de base :**
> AIO-OCIO est une configuration de conversion de couleurs basée sur le PixelManager OCIO original de Genco Uney, optimisée pour les flux de travail de production standard sur plusieurs applications CG et de compositing.
>
> **Prérequis :** Vos applications logicielles doivent prendre en charge la norme **OCIO 2.0** (ou plus récente) pour charger cette configuration.
