# Manuel de l'Industrial AOV Connector

Bienvenue dans le manuel d'utilisation officiel du plugin **Industrial AOV Connector**.

## **Ordre de lecture recommandé**
1. [Mode basique](./basic-mode) — Comprendre la configuration de base et les commandes principales.
2. [Préférences](./preference) — Personnaliser les conventions de nommage, le débruitage et les paramètres de rendu.
3. [Mode avancé](./advanced-mode) — Découvrir les formats de codecs EXR et les couches de DATA indépendantes.
4. [Fake Deep](./fakedeep) — Utiliser la profondeur basée sur les matériaux pour les flux de travail de composition profonde.

> 💡 **Conseil rapide :**
> Ce plugin prend en charge la conservation des nœuds de composition existants et l'édition non destructive. Vous pouvez soit utiliser le bouton `Update Current ViewLayer`, soit activer/désactiver le paramètre `Clear Nodes When Running Cook Nodetree` dans les Préférences pour éviter d'effacer les configurations personnalisées.
