# Welcome to the Industrial 3D Devlog!

Welcome! Today we are officially launching the **Industrial 3D Ecosystem Devlog**. 

This devlog serves as our central hub for sharing deep-dives into:
- 💻 **Cycles Kernel Development**: C++ modifications for rendering performance and custom feature nodes.
- 🔌 **Blender Addon Integrations**: Automation tools like AOV Connector and Light AOV Splitter.
- 🎨 **Color Management**: Advanced OpenColorIO pipeline setups for film and television pipelines.
- ⚙️ **VFX Production Pipelines**: Best practices for integrating Blender into high-end pipeline workflows.

We will keep posts in a unified chronological feed. We're excited to have you along for the ride!
