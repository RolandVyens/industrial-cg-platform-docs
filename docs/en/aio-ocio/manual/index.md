# AIO-OCIO Manual

Welcome to the user manual for **AIO-OCIO** (All in 1 OpenColorIO configuration).

## **Recommended Reading Order**

1. [General Software Setup](./general-softwares) — Integrating with Maya, Nuke, Houdini, and setting system environment variables.
2. [Blender Setup](./blender-setup) — Custom overrides for Blender and setting up the Lin709 workflow.

---

> 💡 **Core Design Concept:**
> AIO-OCIO is a color transform config based on the original PixelManager OCIO by Genco Uney, optimized for standard production workflows across multiple CG and compositing applications.
>
> **Prerequisite:** Your software applications must support the **OCIO 2.0** (or newer) standard to load this configuration.
