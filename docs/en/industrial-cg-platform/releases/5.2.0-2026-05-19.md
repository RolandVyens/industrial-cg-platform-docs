## What's New

### Industrial CG Platform Branding

- Window title now shows `Blender 5.2.0 Industrial CG Platform`.
- Custom splash screen with Industrial CG Platform artwork.
- Custom Windows icons for `.exe` and `.blend` file association.
- CLI and crash reports include the branch suffix.
- Follows Goo Engine methodology: Blender remains the primary product identity.

### ViewLayer Manager (BQt)

- Qt-based ViewLayer management tool integrated as a bundled System Extension.
- Launcher button in Blender's top-right header next to the native ViewLayer selector.
- Features: ViewLayer lifecycle, pass management, Shader AOV, lightgroups, Cycles light-pass AOVs.
- Preset system with save/update/apply/delete for pass configurations.
- Multi-select ViewLayer support for batch preset application.
- Engine-aware section visibility (Cycles vs. Eevee).
- UI translations: Simplified Chinese, Traditional Chinese, French.
- Correct bundled extension repo layout (fixes earlier `system/system/` nesting bug).

### I18N Updates

- Chinese (Simplified and Traditional) and French translations for:
  - Deep EXR UI labels and descriptions
  - Shadow Color UI labels and descriptions
  - Light Pass AOV and Lightgroup Light Pass AOV labels
  - Industrial CG Platform brand suffix

## Known Issues

- **Windows only** — This release is a Windows x64 ZIP package only. Linux and macOS builds are not available.
- **BQt safe-mode log** — The `failed to get blender hwnd, creating new window` message is expected behavior, not a failure.
- **Build system** — Ninja + VsDevCmd is the recommended build path. The Visual Studio generator may hang on some workstations.
