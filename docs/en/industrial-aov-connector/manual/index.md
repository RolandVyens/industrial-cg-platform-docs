> 💡 **Quick Tip:**
> This plug-in supports retaining existing compositor nodes and non-destructive editing. You can either use the `Update Current ViewLayer` button, or toggle the `Clear Nodes When Running Cook Nodetree` setting in the Preferences to prevent erasing custom setups.
