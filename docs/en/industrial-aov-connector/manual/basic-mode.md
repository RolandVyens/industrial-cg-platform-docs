# Basic Mode

This page outlines the basic features and interface options of the **Industrial AOV Connector**.

## **UI Default Content**

### 1. **Main Config**
Pre-made rendering recipes for Basic Mode that control the output layer configurations and compositor structure.

### 2. **Output Settings**
Toggles for quickly adjusting the parameters of nodes that are about to be generated (denoising, custom prefixes, etc.).

### 3. **`Cook Nodetree` Button**
Generates compositor output nodes for **all** viewlayers in the scene. By default, this clears the existing nodes in the compositor before creating the new tree, but you can configure it to keep them via the toggle in the preferences.

### 4. **`Update Current Viewlayer` Button**
Generates nodes for the **currently active viewlayer only**. This is non-destructive and will not affect any existing nodes other than those belonging to the active viewlayer.

### 5. **`Arrange Connector Nodes` Button**
Re-aligns and cleans up the node layout associated with the Industrial AOV Connector inside the compositor editor.

## **UI Hidden / Helper Content**

### **`Delete Useless Default Renders` Button**
Hidden by default. This helper button deletes the folder named `trash_output` containing default Blender renders. This is safe to perform because valid output paths generated by the addon are always placed outside of this folder.

To enable this cleanup button:
1. Click **`Preference`** in the header.
2. Locate the **"Output Tools"** section.
3. Turn on both toggles:
   * **`Default useless renders gather`**: Automatically changes Blender's default render output path to the `trash_output` subfolder.
   * **`Show useless renders clean button`**: Exposes the "Delete Useless Default Renders" button in the Output Tools panel.
