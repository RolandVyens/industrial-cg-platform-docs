# 阴影颜色

<Badge type="tip" text="已发布" />

## 什么是阴影颜色？

阴影颜色（Shadow Color）为 Blender Cycles 中的每个灯光和世界属性添加了颜色属性，用于为各灯光源投射的阴影区域着色。阴影不再只是光的缺失，您可以赋予它们特定的颜色——这是电影和动画灯光中常见的艺术技法。

## 为什么使用它？

- **艺术化控制** — 让阴影的色调完美匹配您的色彩脚本（color script），而无需更改光源本身的颜色。
- **逐灯光的细粒度控制** — 在同一个场景中，不同的灯光可以投射出不同颜色的阴影。
- **世界阴影颜色** — 世界环境照明（World lighting）也可以拥有其独特的阴影颜色。
- **非破坏性** — 阴影颜色仅影响阴影区域，已被照亮的区域保持不变。

## 如何启用？

### 逐灯光阴影颜色

1. 在视口中选择一个灯光。
2. 打开 **属性面板 > 物体数据属性 (Properties > Object Data Properties)**（灯光图标）。
3. 找到 **阴影颜色 (Shadow Color)** 属性。
4. 点击色块选择阴影色调。

### 世界阴影颜色

1. 打开 **属性面板 > 世界属性 (Properties > World Properties)**。
2. 找到 **阴影颜色 (Shadow Color)** 属性。
3. 设置所需的世界阴影色调。

## 艺术用例

| 场景 | 阴影颜色 | 效果 |
| --- | --- | --- |
| 暖色日落场景 | 蓝/紫色阴影 | 与暖光形成冷色对比 |
| 水下场景 | 深青色阴影 | 增强水下深度和水色质感 |
| 风格化动画 | 饱和的互补色 | 营造大胆、图形化的阴影效果 |
| 月光 | 深蓝色阴影 | 经典的夜间户外冷夜调感觉 |

::: tip
建议从微妙、低饱和度的阴影颜色开始。一点点色调就能产生显著的变化。该效果在软阴影衰减（soft shadow falloff）的过渡区域中最为明显。
:::

## 已知限制

- 阴影颜色仅影响该特定灯光的阴影贡献，不会改变场景中其他灯光的阴影行为。
- 该效果在阴影区域纯粹是加色渲染（additive color），并不会模拟彩色半透明遮光物体的折射阴影。
