# 发布说明 5.2.0 (2026-05-19)

**代码分支:** `industrial-cg-platform`

**编译分支 (Build branch):** `industrial-cg-platform`

## 新增特性说明

### Industrial CG Platform 深度品牌定制

- 运行时主窗口标题现在正式显示为 `Blender 5.2.0 Industrial CG Platform`。
- 深度定制的启动画面 (Splash Screen)，采用 Industrial CG Platform 艺术原画。
- 附带针对 `.exe` 和 `.blend` 文件后缀关联的自定义 Windows 图标。
- 命令行界面 (CLI) 以及闪退崩溃日志报告中，都集成了该生产分支的后缀标识。
- 遵循经典的 Goo Engine 定制规范：Blender 仍然作为最核心的平台与软件身份。

### ViewLayer 管理器 (BQt)

- 作为捆绑自带的系统扩展集成，提供了独立的、基于 Qt 的高级 ViewLayer 管理视窗。
- 启动按钮清晰地放置在 Blender 右上角标题栏中，紧邻原生的 ViewLayer 下拉选择菜单。
- 核心功能：ViewLayer 完整生命周期管理、渲染通道复选、Shader AOV 管理、灯光组配置、Cycles 灯光通道分量 AOV 控制。
- 提供了高度易用的预设系统，支持对复杂的通道组合进行保存 (Save)、更新 (Update)、应用 (Apply) 和删除 (Delete) 操作。
- 支持在左侧列表中按住 Ctrl/Shift 多选 ViewLayer，一键进行批量的通道预设应用。
- 界面可自适应当前激活的渲染引擎，智能过滤 Cycles 和 Eevee 所专属的设置选项。
- UI 交互界面完整支持简体中文、繁体中文和法语本地化翻译。
- 修复了早期版本因 `system/system/` 文件夹嵌套错误导致扩展无法正常扫描的 Bug。

### I18N 汉化与国际化

- 为以下核心底层新增了完整的中文（简体/繁体）及法语的系统翻译支持：
  - Deep EXR 相关的 UI 交互标签与系统属性描述
  - 阴影颜色（Shadow Color）属性标签与描述
  - 灯光通道 AOV 和灯光组分量通道 AOV 的详细标签
  - Industrial CG Platform 专属的产品后缀字串

## 已知局限与问题 (Known Issues)

- **仅限 Windows 系统** — 该发布包当前仅提供 Windows x64 ZIP 格式免安装压缩包，Linux 和 macOS 平台的二进制构建暂时不可用。
- **BQt 安全模式警告** — 启动时在终端打印的 `failed to get blender hwnd, creating new window` 警告属于正常的设计预案，并不代表任何功能失败或不可用。
- **配置编译** — 强烈建议在编译时使用 Ninja + VsDevCmd 组合。部分 Windows 开发工作站上的 Visual Studio 默认 CMake 生成器在探测编译器时可能会发生挂起。
