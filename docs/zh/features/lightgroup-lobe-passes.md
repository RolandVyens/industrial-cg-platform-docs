# 灯光组分量通道

<Badge type="tip" text="已发布" />

## 简介

灯光组分量通道扩展了 Blender Cycles 的灯光 Pass 系统，将每个**灯光组**分解为其各个照明分量（lobe）。您现在可以为每个灯光组输出单独的**漫反射**、**光泽**、**透射**和**体积**通道，并进一步分离**直接光**和**间接光**。

## 为什么使用它

- **精细重打光** — 在合成中按分量类型调整各个灯光贡献。
- **逐灯光组分解** — 精确查看每个灯光组如何独立贡献漫反射、光泽、透射和体积。
- **聚合平衡** — 所有逐灯光组分量通道的总和可还原回合并的 Beauty Pass。
- **生产验证** — 已在真实生产场景中使用面光、点光、聚光灯和太阳光类型验证。

## 如何启用

1. 打开 **属性 > 视图层属性 > Pass > 灯光**。
2. 正常创建灯光组。
3. 勾选 **Light Pass AOVs** 启用逐灯光组分量 Pass 输出。
4. 选择所需的分量组件。

## 输出命名规则

每个灯光组分量 Pass 按以下模式命名：

```
<分量>_<Direct|Indirect>_<灯光组名称>
```

例如，灯光组名为 `key` 时：

| Pass 名称 | 内容 |
| --- | --- |
| `Combined_key` | 合并灯光组 Pass |
| `Diffuse_Direct_key` | `key` 灯光的直接漫反射 |
| `Glossy_Direct_key` | `key` 灯光的直接光泽 |
| `Transmission_Direct_key` | `key` 灯光的直接透射 |

## 聚合平衡

分量通道设计保证：

```
Combined_<lg> ≈ Σ (分量_Direct_<lg> + 分量_Indirect_<lg>)
```

## 已知限制

- **自发光网格** — 自发光网格灯光组保持仅合并模式，不分解为直接/间接分量。
- **完整 LPE** — 完整 Light Path Expression 语法是未来工作。
