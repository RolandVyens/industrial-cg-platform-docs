# 阴影颜色

<Badge type="tip" text="已发布" />

## 简介

阴影颜色为 Blender Cycles 中的每个灯光和世界添加了颜色属性，用于为各灯光源投射的阴影区域着色。阴影不再只是光的缺失，您可以赋予它们特定的颜色——这是电影和动画灯光中常见的艺术技法。

## 如何启用

### 逐灯光阴影颜色

1. 在视口中选择一个灯光。
2. 打开 **属性 > 物体数据属性**（灯光图标）。
3. 找到 **阴影颜色** 属性。
4. 点击色块选择阴影色调。

### 世界阴影颜色

1. 打开 **属性 > 世界属性**。
2. 找到 **阴影颜色** 属性。
3. 设置所需的世界阴影色调。

## 艺术用例

| 场景 | 阴影颜色 | 效果 |
| --- | --- | --- |
| 暖色日落场景 | 蓝/紫色阴影 | 与暖光形成冷色对比 |
| 水下场景 | 深青色阴影 | 增强深度和水色 |
| 风格化动画 | 饱和的互补色 | 大胆的图形化阴影效果 |
| 月光 | 深蓝色阴影 | 经典夜间外景感觉 |

## 已知限制

- 阴影颜色仅影响特定灯光的阴影贡献，不会改变场景中其他灯光的阴影行为。
