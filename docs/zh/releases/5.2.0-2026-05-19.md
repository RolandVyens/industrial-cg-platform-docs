## 新增功能

### Industrial CG Platform 品牌标识

- 窗口标题现在显示 `Blender 5.2.0 Industrial CG Platform`。
- 带有 Industrial CG Platform 原画的自定义启动画面。
- 针对 `.exe` 和 `.blend` 文件关联的自定义 Windows 图标。
- CLI 和崩溃报告包含分支后缀。

### ViewLayer 管理器 (BQt)

- 作为捆绑的系统扩展集成的基于 Qt 的 ViewLayer 管理工具。
- Blender 右上角标题栏中原生 ViewLayer 选择器旁边的启动按钮。
- 功能：ViewLayer 生命周期、通道管理、Shader AOV、灯光组、Cycles 灯光通道 AOV。
- 带有保存/更新/应用/删除功能的通道配置预设系统。
- 引擎感知的部分可见性（Cycles 与 Eevee）。
- UI 翻译：简体中文、繁体中文、法语。

### I18N 更新

- 针对以下内容的中文（简体和繁体）及法语翻译：
  - Deep EXR UI 标签和描述
  - 阴影颜色 UI 标签和描述
  - 灯光通道 AOV 和灯光组灯光通道 AOV 标签
  - Industrial CG Platform 品牌后缀
